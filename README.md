# qckdev.Text.Json
 
Provides types that serialize and deserialize JSON using System.Text.Json for more recent frameworks and Newtonsoft.Json for older ones transparently.

**System.Text.Json**
  - .NET Standard 2.0 

**Newtonsoft.Json**
  - .NET Framework 4.6.1
  - .NET Standard 1.2
